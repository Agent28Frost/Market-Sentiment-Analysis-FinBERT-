# Market Sentiment Analysis (FinBERT)

AI-powered news sentiment analysis for futures markets (ES, NQ, GC, CL).

## Features
- FinBERT-based sentiment
- News aggregation
- CSV export
- Market bias detection

## Setup
pip install -r requirements.txt
python sentiment_analysis.py

## Example Output
(Bullish / Bearish summary)
